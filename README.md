# Serif Generative Math Lab

This folder contains the expanded interactive math and physics lab for Serif.

## Open

Open `index.html` in a browser. The page is a single self-contained HTML file, with external CDN links for fonts, KaTeX, and Three.js.

## Contents

- `index.html` - the current enhanced page.
- `originals/index_2_original_backup.html` - the original file before the math-lab expansion.
- `dev/enhance_index2.py` - the generator script used to rebuild the enhanced page from the original backup.
- `previews/` - screenshots from browser verification.

## Notes

The enhanced page includes the original four generators plus the expanded equation lab, 3D systems, and play controls for visible motion.
